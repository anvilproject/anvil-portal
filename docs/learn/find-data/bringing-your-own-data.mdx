Users can also upload their own datasets to an AnVIL Terra workspace and use Terra's sharing mechanisms to manage and control access. 

## Learn more! Terra Support Resources      
- For more details, see [Best practices for sharing and protecting data resources](https://support.terra.bio/hc/en-us/articles/360042670651-Best-practices-for-sharing-and-protecting-data-resources)     
- See [How to move data between local and workspace storage](https://support.terra.bio/hc/en-us/articles/360058268832-How-to-move-data-between-local-and-workspace-storage) for step-by-step instructions    
- See [Managing data with tables](https://support.terra.bio/hc/en-us/articles/360025758392-Managing-data-with-tables) to learn how to organize and manage data in Terra    
- See [How to upload data and populate a data table with linked file paths](https://support.terra.bio/hc/en-us/articles/4419428208411-Upload-data-and-populate-the-table-with-linked-file-paths)          
- See [Sharing data and tools with workspace access controls](https://support.terra.bio/hc/en-us/articles/360025851892-Sharing-data-and-tools-with-workspace-access-controls) to understand how to protect access to your data                 

Terra's access controls can also be leveraged to protect pre-release datasets during upload and [QC](/learn/data-submitters/submission-guide/ingesting-data#42-validation-steps-automated).
